# Il Ricettario

Applicazione web per la gestione e condivisione di ricette, sviluppata con Node.js, Express ed EJS, con database SQLite.

**Video di presentazione:** https://youtu.be/UrP76ROgv8E

---

## Requisiti

- [Node.js](https://nodejs.org/) v18 o superiore

---

## Installazione e avvio

```bash
# 1. Installare le dipendenze
npm install

# 2. Avviare il server
npm start 
oppure
node ./bin/www
```

Aprire il browser su **http://localhost:3000**
